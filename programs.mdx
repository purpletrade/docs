## Program Architecture

```
┌──────────────────────┐     ┌──────────────────────┐
│  Percolator Program  │     │   Matcher Program     │
│  (perpetual engine)  │────▶│   (LP matching)       │
└──────────┬───────────┘     └───────────────────────┘
           │
           ▼
┌──────────────────────┐
│    Slab Account      │
│  (full market state) │
│  4096 account slots  │
│  ~992 KB, ~6.9 SOL   │
└──────────────────────┘
```

Each perpetual market is a single slab account owned by the Percolator program. The Matcher program handles LP order matching via CPI (cross-program invocation).
